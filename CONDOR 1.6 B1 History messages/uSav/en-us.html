Cyber Data Security and Sharing

Select a file to start uSav.

File selection can be done by:

1/ Tap on a file in uSav folders, or

2/ Press on a file in any App, or

3/ Select a file in any App and choose uSav App in the Open-in feature.

uSav allows you to secure and share your files through cloud storage, email, USB memory stick or on any storage anywhere.  

It uses AES 256 encryption with one unique encryption key per file, which is the protection also used by the US government.  Another unique feature is that even after you have shared the file, you can change its access permission anytime to prevent the user from opening it.   Each protected file has a File History/Audit Trail showing who has accessed the file and when.

Here is a brief description of the functions:

  • Encrypt and Share  allows you to quickly encrypt a file and share it with others via email
  • Decrypt and view allows you to quickly decrypt a file from others and preview it 
  • Advanced Mode gives you more control on the following:
    • File allows you to encrypt or decrypt all the files on the device, and see what have been encrypted and decrypted 
    • Contact allows you to manage your contacts and groups
    • History allows you to view operation history like what operation you have done, and file access history like who has open which file and when
    • Profile allows you to change password, get help or give us feedback
