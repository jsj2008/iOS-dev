Secure File Sharing and Storage

When you share a file via email or cloud drive, or store a file in the cloud drive, the file is exposed to hackers and service provider staffs.  

uSav can protect your files!  It uses the strongest protection available (AES-256) and allows you to share it via email, cloud drive, or any way you want.  With uSav, you can share a photo from your photo library, or a file imported from another app or email, with the assurance that no one else can see that file.

This app is easy to use:

  • Quick Encrypt allows you to quickly encrypt a file and share it with others via email
  • Quick Decrypt allows you to quickly decrypt a file from others and preview it 
  • Advanced Mode gives you more control on the following:
    • File allows you to encrypt or decrypt all the files on the device, and see what have been encrypted and decrypted 
    • Contact allows you to manage your contacts and groups
    • Logs allows you to view the history and activity logs
    • Profile allows you to change password, get help or give us feedback
